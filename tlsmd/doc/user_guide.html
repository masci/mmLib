Animating TLS Models

The script, "pdb_animate.pl" (found in the tarball for every job run through TLSMD), can be used to generate animated GIFs of the TLS described modeled PDBs moving through 8 phases of prediction libration.

To use the script, one must take the output PDB (e.g., "XXXX_CHAINC_NTLS4_ANIMATE.pdb") for a given partition of a given chain and its associated HTML description page (e.g., "XXXX_CHAINC_NTLS4_ANIMATE.html") and use them as the input for the pdb_animate.pl script. Both of these files are located in the tarball of every job run through TLSMD.

Below is the syntax of this script:

./pdb_animate.pl <input_ANIMATE.pdb> <input_ANIMATE.html> <chain_id>

DESCRIPTION:
TLS Minimal Domains (TLSMD) is a program which computes optimal 
TLS (rigid-body) descriptions from the isotropically or anisotropically 
refined ADPs of a protein crystal structure. 

REQUIREMENTS:
  * pymmlib >= 1.1.0

INSTALLATION:
Read INSTALL.txt or visit http://skuld.bmsc.washington.edu/~tlsmd/install.html

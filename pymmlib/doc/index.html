Python Macromolecular Library Documentation

The Python Macromolecular Library (mmLib) is a programming library for writing Python scrips to manipulate macromolecular structure files, load macromolecular structures into a hierarchical data structure, and inspect the structure's model parameters. The macromolecular data structure supports crystallographic models and NMR models, and has additional modules to for calculating unit cell parameters and symmetry operators associated with the crystal structure.
